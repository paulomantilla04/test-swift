#  Swift Project

This project implements reusable interface components in SwiftUI, focusing on profile card designs with custom styling and entrance animations.

## Features

- Reusable Components: Implementation of `CustomCard` ensure visual consistency across the application.
- Dynamic Design: Utilizes linear gradients, continuous corner radiuses, and shadows to achieve a modern aesthetic.
- Animations: Entrance effects using opacity and offset states triggered during the component's lifecycle via .onAppear.
- Media Integration: Support for both local image assets and SF Symbols for iconography.

## Requirements
- Xcode 15.0 or higher.
- iOS 17.0+ (recommended for previews and Swift 5.0+ syntax).

Installation
- Clone the repository.
- Open `testProject.xcodeproj` in Xcode.
- Select an iPhone simulator and press `Command + R` to run the project.

## Technologies Used
- Language: Swift 5.0.
- Framework: SwiftUI.
- Layout Management: Adaptive spacing using VStack, HStack, and ZStack.
